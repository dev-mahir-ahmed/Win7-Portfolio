💻 Windows 7 Inspired Web Portfolio

This is a fully interactive desktop-style portfolio web application designed to replicate the experience of using a Windows 7 operating system inside the browser.

The project transforms a traditional portfolio into an immersive desktop environment where users can explore information through familiar OS elements.

✨ Features
🖥️ Windows 7–like desktop interface
📁 My Computer, Projects, Resume, Terminal
🧮 Built-in Calculator, Notepad, Paint simulation
🖱️ Right-click context menu (Refresh, New, Personalize, etc.)
📂 Window-based multitasking system
🎨 Fully interactive UI/UX experience
⚡ Smooth animations and OS-like behavior
🎯 Purpose

To create a unique and engaging way of presenting a developer portfolio using an operating system simulation concept.

🚀 Tech Stack
Html • CSS • Javascript
